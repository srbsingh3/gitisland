# GitIsland

**Your GitHub contributions in macOS Dynamic Island**

A minimal menu bar app that displays your GitHub contribution graph in a sleek, expandable notch overlay.

https://github.com/user-attachments/assets/09ea2030-1044-49af-9b54-e7f407725bf4

## Features

- **Dynamic Island UI** — Expands on hover with smooth spring animations
- **Live GitHub Data** — Fetches your real contribution history via GraphQL
- **Minimal & Native** — Pure SwiftUI, no dependencies

## Setup

1. Clone and open in Xcode
2. Add your `GITHUB_TOKEN` to the scheme environment variables
3. Build and run
