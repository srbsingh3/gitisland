## Features

| | |
|---|---|
| **Dynamic Island UI** | Expands on hover with smooth spring animations |
| **Live GitHub Data** | Fetches your real contribution history via GraphQL |
| **Minimal & Native** | Pure SwiftUI, no external dependencies |

<br>

## Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/gitisland.git

# Open in Xcode
open gitisland.xcodeproj
```

Then add your `GITHUB_TOKEN` to the scheme's environment variables and run.
